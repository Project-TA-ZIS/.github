# Project TA - Monitoring Dasawisma & ZIS

<p align="center">
  <img src="team-pict.jpg" alt="Team Project" width="700" />
</p>

Sistem informasi untuk monitoring dan pengelolaan keuangan Dasawisma serta ZIS
(Zakat, Infaq, Shodaqoh). Project ini terdiri dari aplikasi backend REST API dan
frontend SPA.

## Ringkasan Aplikasi

- **Backend**: `Backend_TA` - Node.js, Express, Sequelize, MySQL/MariaDB, Swagger, Jest, dan k6.
- **Frontend**: `Frontend-TA` - React, Vite, Tailwind CSS, Zustand, Axios, dan Vitest.
- **Dokumentasi profil**: `profile/profile` - README project-level dan gambar tim.


## Prasyarat

- Node.js versi LTS
- MySQL/MariaDB
- npm
- k6, jika ingin menjalankan performance test
